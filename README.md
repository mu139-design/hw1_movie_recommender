# Movie Recommendation System

This project implements a prototype Movie Recommendation System developed for the Rutgers Data Management course.

The system analyzes movie genres and user ratings to determine movie popularity, genre trends, and personalized recommendations.

The program processes movie and rating datasets and provides recommendations based on a user's most preferred genre.

---

## Features

The program implements the following functionality:

- Compute movie popularity based on average ratings
- Display Top N movies overall
- Display Top N movies within a genre
- Compute genre popularity
- Display Top N genres
- Determine a user's most preferred genre
- Recommend movies based on a user's preferred genre

Recommendations are generated by:

1. Identifying the user's highest-rated genre  
2. Selecting the most popular movies within that genre  
3. Excluding movies the user has already rated

## Project Files

movie_recommender.py  
The main program that implements the recommendation system and provides a command line interface (CLI).

test_movie_recommender.py  
Automated testing script used to verify that all functions behave correctly.

movies.txt  
ratings.txt  
Example datasets used for testing the system.

---

## Running the Program

Run the main recommendation program:

python movie_recommender.py

This launches the CLI menu where you can:

- Load movie data
- Load rating data
- View movie rankings
- View genre popularity
- Generate movie recommendations

---

## Running the Automated Tests

Run the automated test program:

python test_movie_recommender.py

The tester loads input datasets and calls each function to verify the correctness of the system outputs.

---

## Input Data Format

Movies File Format

genre|movie_id|movie_name

Example:

Adventure|1|Toy Story (1995)

Ratings File Format

movie_name|rating|user_id

Example:

Toy Story (1995)|4.0|1

Ratings must be within the range 0–5.
